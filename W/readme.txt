Using projectStarter create a new folder called "W3A" in your Assignments. 
Go through the Git/Github steps to connect the repositories
Run your live-server to serve your site
In your index.html file add the following content:
A header that contains the name of your site and a small welcome message to the user
A main that contains 3 articles, each article contains:
The name of an animal
An image of that animal
A small description of that animal (can just be grabbed from wikipedia)
Connect a CSS file to index.html
Add the following selectors to change any CSS that you would like on the page:
Descendant selector (space)
Child selector ( > )
Adjacent selector ( + )
:hover
:active
